## Smart Accountant

Smart Accountant is a web application Made With Laravel It allows you to Manage Multiple Marketplace

- [Easy Install]
- [Simple]
- [Full Control]

## 1. Setting Up 

1. Clone The Project:
 ```sh
   git clone https://github.com/ahmed1amen/SmartAccountant.git
   ```

2. Run Composer:
 ```sh
   Composer install
   Composer Update
   ```
3. Run npm :
 ```sh
   npm install
   npm run dev
   ```
